# TDE-01-WEB-DEV
atividade de web devlopment

Funções e suas explicações resumidas:

filtrarPares(array) → recebe um array e retorna apenas os números pares (usando o método .filter()).

calcularMedia(array) → soma todos os valores do array (com .reduce()) e divide pela quantidade de elementos para achar a média

executarOperacao(a, b, operacao) → recebe dois números e uma função de operação (como soma, subtracao etc).

mostrarResultado(callback, operacao) → pega os números digitados na tela, chama a função desejada e mostra o resultado.

obterArray() → lê o campo de texto e transforma “1,2,3,4” em [1, 2, 3, 4].

filtrarArray() → mostra os números pares digitados.

mostrarMedia() → mostra a média dos números digitados
